⌨️  KEYBOARD SHORTCUTS:

Navigation:
  F             Focus all nodes (fit to view)
  R             Reset zoom
  +/-           Zoom in/out

Layout & Display:
  L             Toggle Force ↔ Hierarchical layout
  Space / M     Toggle minimap

Search:
  Ctrl/Cmd + F  Focus search box
  Esc           Clear search

Help:
  ?             Show/hide keyboard help
