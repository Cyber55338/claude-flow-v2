The Claude Flow terminal panel is now fully interactive!

┌──────────────────────────────────────────────────────────────┐
│  HOW TO USE                                                   │
└──────────────────────────────────────────────────────────────┘

1. Open browser at: http://localhost:3000

2. Look at the LEFT PANEL (Terminal section)

3. You'll see an input box at the bottom with:
   $ [Type your message to Claude Code...]

4. Type any message and press ENTER

5. Watch the nodes appear on the RIGHT canvas instantly!


┌──────────────────────────────────────────────────────────────┐
│  TRY THESE COMMANDS                                           │
└──────────────────────────────────────────────────────────────┘

hello           - Get a greeting response
help            - See available commands
test            - Generate test nodes
metacognitive   - See metacognitive flow nodes appear
clear           - Clear terminal (Ctrl+L also works)


┌──────────────────────────────────────────────────────────────┐
│  KEYBOARD SHORTCUTS IN TERMINAL                               │
└──────────────────────────────────────────────────────────────┘

Enter       Send message
↑ ↓         Navigate command history
Ctrl+L      Clear terminal
Ctrl+Z/Y    Undo/redo (global)


┌──────────────────────────────────────────────────────────────┐
│  WHAT HAPPENS                                                 │
└──────────────────────────────────────────────────────────────┘

Type: "hello"
  ↓
Terminal shows your message
  ↓
"Claude" responds
  ↓
Parser creates nodes
  ↓
Nodes appear on canvas INSTANTLY via WebSocket!


┌──────────────────────────────────────────────────────────────┐
